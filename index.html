CSS Animation

The animation property in CSS makes it possible to animate many other CSS properties from one CSS style configuration to another. CSS animation is an effort to animate components on a web page without using processor and memory-hungry JavaScript or Flash. Each animation is composed of two components, @keyframes that indicate the start and end states, as well as possible intermediate waypoints which is then described in a CSS style to be called upon some element. A very basic example of CSS animation would look something like below: 


.element {
    animation: pulse 5s infinite;
}

@keyframes pulse {
    0% {
    background-color: #F8F8F8;
    }
    100% {
    background-color: #323131;
    }
}

On above example @keyframes defines states at specific moment of the animation. For example, the beginning of the animation is expressed by 0% and 100% is marking the end. CSS animation property has several additional sub-properties which gives more control over how those keyframes are being managed. These eight sub-properties can be used as shorthand too.

These sub-properties and shorthand can be used interchangeably: 


@keyframes stretch {
    /* declare animation actions here */
}

/*
    using sub-properties:
*/

.element {
    animation-name: stretch;
    animation-duration: 3s; 
    animation-timing-function: ease-out; 
    animation-delay: 1s;
    animation-direction: alternate;
    animation-iteration-count: infinite;
    animation-fill-mode: none;
    animation-play-state: running; 
}

/*
    is the same as above using shorthand:
*/

.element {
    animation: 
        stretch
        3s
        ease-out
        1s
        alternate
        infinite
        none
        running;
}

The sub-properties of the animation property are:

  • animation-name: describes the name of the @keyframes at-rule specifying the animation's keyframes.
  • animation-duration: specifies the length of time for an animation to complete one cycle.
  • animation-timing-function: configures how animation transitions through keyframes with acceleration curves like linear or ease.
  • animation-delay: the length of time between the element is being loaded and the start of the animation sequence.
  • animation-iteration-count: defines the number of times the animation should be performed. Specifying { animation-iteration-count: infinite; } would repeat the animation indefinitely.
  • animation-direction: configures the direction of the animation. The defaults resets on each cycle.
  • animation-fill-mode: sets the values are applied before and after the animation.
    For example, it is possible to set the last state remain on screen after the animation or switch back to the state that was before the animation began.
  • animation-play-state: determines the paly and pause state of the animation.
    For example, setting the value running would keep the animation running.

CSS animation property can be used to animate many other CSS properties like color, height or width.

The above is a great example of complex CSS animation in action. In this example there are 2 named animation 'lower' and 'upper' described the keyframes at multiple point of time. The animation named upper is responsible for the animation of the first line with text MYKE and another is for MELO . In both of those animations the opacity and text-shadow property is being described several points of the animation cycle. The animations will run indefinitely with linear transition through keyframes. The duration of the upper animation is 11s and lower one is 10s to stage a scenario of incongruity. The interesting thing to notice here is that the animations are not being applied on the whole text but a portion.

Based on popular website Can I Use?, the support for CSS Animation across desktop and mobile web browser is very good. It has a 95.42% Global support as of this writing. All major browsers available regardless of desktop/laptop or mobile support most of the features with great stability.

CSS animation is an incredibly useful and powerful tool which can add interest or creative excitement, catch user's eye or explain things quickly and succinctly. This is the reason that the usage of CSS animation is growing rapidly on sites and in app.
